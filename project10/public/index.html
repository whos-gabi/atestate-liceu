Absorbtia intestinala

Absorbtia intestinala este trecerea produsilor rezultati in urma digestiei catre sânge si limfa.

Absorbtia este favorizata la nivelul intestinului subtire, deoarece:

  • suprafata de contact este mare
  • distanta pe care moleculele o au de strabatut este mica
  • reteaua vasculara de la nivelul vilozitatilor este foarte bogata
  • miscarile contractile ale vilozitatilor inlesnesc tranzitul substantelor absorbite

ABSORBTIA GLUCIDELOR: produsii finali sunt

  • GLUCOZA
  • GALACTOZA
  • FRUCTOZA

ABSORBTIA PROTEINELOR: produsi finali sunt
AMINOACIZII -> se absorb activ

ABSORBTIA LIPIDELOR:

- prin difuziune pasiva;
- lipide + saruri biliare = hidrosolubile

  • AMINOACIZI
  • MONOZAMARIDE
  • LIPIDELE

Digestia, absorbtia si excretia la nivelul intestinului gros

ACTIVITATEA MOTORIE LA NIVELUL INTESTINULUT GROS:

  • absorbtia apei si a electrolitilor
  • depozitarea materiilor fecale
  • miscarile de la nivelul colonului:
    • miscari de amestec
    • miscari propulsive

ABSORBTIA SI SECRETIA LA NIVELUL COLONULUR:

  • apa: 2-3 I/zi la nivelul colonului
  • Nat si CI- -> absorbite de colon
  • secretà K+

DEFECATIA = procesul de eliminare a materiilor fecale din intestin.
